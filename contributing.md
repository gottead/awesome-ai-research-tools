# Contributing

Contributions are very welcome — this list is only useful if it stays current.

## Adding a tool

Open an [issue](https://github.com/0x11c11e/awesome-ai-research-tools/issues) or a pull request with the tool name, URL, and a one-line factual description (under 18 words).

Please make sure the tool:

- Is **alive** — the link resolves and the product is actively available.
- Is **research-focused** — built for academic or scientific research. General-purpose chatbots are out of scope.
- Is **distinct** from what's already listed.

## Formatting

- Place the tool in its most specific category, in alphabetical order.
- Use the format: `- [Name](https://example.com) - Description.`
- Start the description with a capital letter and end it with a period.
- Avoid marketing language ("revolutionary", "powerful", "next-gen") — describe what the tool actually does.

## Removing a tool

Spotted a dead link or a tool that has pivoted away from research? Open an issue and it will be removed in the next sweep.
